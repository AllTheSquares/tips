Random Tips

Hosting your own website on GitHub

  1. Begin by creating a new repository. Name it as followed: "username".github.io (Replace "username" with your GitHub username, you can see this in the web address of this page.
  2. I recommend that you download GitHub Desktop at this point. It makes editing your site easier than having to publish changes via a terminal. Once downloaded, clone your repository via this app.
  3. Cloning a repository creates a folder on your computer. On your favorite code editor of choice (I recommend either Visual Studio Code or Sublime Text, which may need a license to use), create a new file named "index.html amd save it to the folder of your repository on your computer (should be in the GitHub folder under Documents).This is your main page.
  4. On GitHub Desktop, it should automatically recognize any changes made to your folder
